Install RatScape App

Chrome / Edge

  1. Click the Install button below
  2. Or look for the Install icon in the address bar
  3. Click "Install" to add to your apps

Firefox

  1. Click the menu button (three lines)
  2. Select "Install" from the menu
  3. Confirm installation

Safari

  1. Click File in the menu bar
  2. Select "Add to Dock" or "Add to Home Screen"
